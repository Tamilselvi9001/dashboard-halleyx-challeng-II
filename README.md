# dashboard-halleyx-challeng-II# 📊 Sales Analytics Dashboard (Halleyx Challenge-II)

A modern, responsive web application designed to manage customer orders and provide real-time business insights through interactive charts.

## ✨ Key Features
* **Order Tracking**: View a complete list of customer orders with details like Product Name, Amount, and Status.
* **Data Visualization**: Real-time Bar Charts for revenue by product and Pie Charts for order status distribution.
* **Smart Fallback**: Automatically switches to LocalStorage if the Backend API is unreachable.
* **Clean UI**: Minimalist design with a focus on data readability and mobile responsiveness.

## 🛠️ Tech Stack
* **Frontend**: HTML5, CSS3, JavaScript (ES6+), Chart.js.
* **Backend**: Python, FastAPI.
* **Server**: Uvicorn.
* **Security**: Handled CORS and local origin security for seamless data flow.

## 🚀 Getting Started

### 1. Run the Backend
Ensure you have Python installed, then run the server:
```bash
python main.py
```
